# Elemes Food

Elemes Food — адаптивный лендинг для сайта еды, созданный с использованием HTML и SCSS.

## Технологии

- HTML5
- SCSS
- CSS3
- Flexbox
- CSS Grid
- Media Queries

## Возможности

- Адаптивный дизайн
- Современный интерфейс
- Hero секция
- Категории блюд
- Популярные блюда
- Footer с контактами
- Mobile меню

## Структура проекта

project/

│ └── style.css
│
├── images/

│
└── README.md

## Запуск проекта

1. Открыть проект в VS Code.
2. Установить расширение Live Sass Compiler.
3. Открыть файл style.scss.
4. Нажать кнопку Watch Sass.
5. Запустить index.html через Live Server.

- Mixins
